# IntelliFit: AI-Powered Home Workout Recommendation System
Capstone Project for SPCBA - Varsity Students.

## Tech Stack
- **Backend:** Python (Flask)
- **Database:** Firebase Firestore & Authentication
- **Frontend:** HTML5, Tailwind CSS
- **Platform:** Mobile-responsive Web App

## Features
- AI-based exercise recommendations
- Athlete Profiling (Sport & Goal)
- Discipline Tracking (Workout Logs)
- Real-time History sync
